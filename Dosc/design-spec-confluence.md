# 1. 메인 허브 (Main Hub) — 은거지(HideOut)

## 1-1. 전체 UI 레이아웃 개요

메인 허브 화면은 **3D 배경 씬(Scene) 위에 HUD 레이어를 오버레이하는 구조**로 구성된다.
UI는 크게 5개 영역(Zone)으로 분리된다.

```
+-----------------------------------------------------------------------------+
|  [ZONE-A] 상단 탭 바                                            [h: 48px]   |
|                                                                             |
|  [홈] 은거지  닉네임·Lv  퀘스트  공방  교환소  성장  성장트리                     |
|  우편함  용병  소식  시스템        [실버]  [D토큰]  [계곡의 인장]  [서버]         |
|                                                                             |
+------------------+----------------------------------+----------------------+
|                  |                                  |                      |
| [ZONE-B]         |         [ZONE-C] 중앙 3D 씬       | [ZONE-D]             |
| 중앙 좌측 패널    |   "PlayerName"  Lv.47            | 중앙 우측 패널        |
| (190px)          |         [3D 캐릭터 렌더]           | (190px)              |
|                  |       (화면을 채우는 크기)          |                      |
| - 퀘스트 추적기   |                                  | - 스쿼드 설정         |
| - 채팅창         |                                  |   (랜덤 스쿼드 포함)  |
|                  |                                  | - 플레이 버튼         |
+------------------+----------------------------------+----------------------+
|  [ZONE-E] 하단 탭 바 (항상 표시 · z-index 최상위)                 [h: 46px] |
|  보관소  캐릭터 변경                                      친구              |
+-----------------------------------------------------------------------------+
```

### [ZONE-C] 중앙 3D 캐릭터 디스플레이

#### 캐릭터 크기
- 내 캐릭터: 화면을 채우는 크기 (기준 100%)
- 좌우 파티원: 내 캐릭터 대비 90% 크기

#### 캐릭터 닉네임/레벨 표기
- 3D 캐릭터 렌더 영역 **상단**에 표기 (캐릭터 머리 위가 아님)
- 스쿼드 파티장은 닉네임 옆에 [왕관] 아이콘 표시

**솔로 (1인)**
```
  PlayerName  Lv.47
  +--------------------+
  |  [3D 캐릭터 렌더]  |
  +--------------------+
```

**스쿼드 (2인)**
```
  [왕관] Teammate_01 Lv.23   PlayerName Lv.47
  +-------------+             +---------------+
  |  [파티원]   |             |  [내 캐릭터]  |
  |   (90%)     |             |   (100%)      |
  +-------------+             +---------------+
```

**스쿼드 (3인)**
```
  [왕관] Teammate_01 Lv.23   PlayerName Lv.47   Teammate_02 Lv.31
  +-------------+            +---------------+   +-------------+
  |  [파티원]   |            |  [내 캐릭터]  |   |  [파티원]   |
  |   (90%)     |            |   (100%)      |   |   (90%)     |
  +-------------+            +---------------+   +-------------+
```

**닉네임 레이블 세부 스펙**

| 속성 | 값 | 비고 |
|---|---|---|
| 표시 위치 | 캐릭터 렌더 영역 상단 | — |
| 표시 내용 | `player.displayName` + `Lv. {level}` | — |
| 파티장 표시 | 닉네임 좌측 왕관 아이콘 | `member.isLeader === true` 시 |
| 폰트 크기 | 내 캐릭터 14px, 파티원 12px | — |
| 텍스트 색상 | 내 캐릭터 `#FFFFFF`, 파티원 `#7EC8E3` | — |
| 배경 | 반투명 검정 `rgba(0,0,0,0.45)`, 라운드 4px | — |
| 최대 닉네임 길이 | 12자 초과 시 말줄임 처리 | — |

### [ZONE-D] 중앙 우측 패널

자동 스쿼드 설정 위젯과 플레이 버튼으로 수직 분할한다.

#### D-1. 스쿼드 설정 위젯

| 항목 | 내용 |
|------|------|
| 스쿼드 ON/OFF | 토글 버튼 · 기본 ON |
| 인원 선택 | 솔로 / 2인 / 3인 |
| 랜덤 스쿼드 | ON/OFF 토글 · 스쿼드 ON + 2인 이상일 때만 노출 |
| 스쿼드 슬롯 | 파티원 목록 표시 (랜덤 시 "랜덤 매칭 대기 중..." 표시) |

#### D-2. 플레이 버튼

플레이 버튼 클릭 → **지역 맵 던전 선택 오버레이** 오픈 (핀 방식)

던전 핀 위험도별 색상:
- LOW: 초록 (#46c878)
- MED: 노랑 (#d4b830)
- HIGH: 주황 (#d86020)
- EXTREME: 빨강 (#d83048)

## 3. 작업장 (Workshop)

진입: ZONE-A 공방 탭 → HUD 오버레이 (사이드바 + 상세 패널 구조)

### 시설 목록

| 시설 | 최대 레벨 | 기능 |
|------|----------|------|
| 기본 작업대 | Lv.3 | 기초 아이템 및 부품 제작 |
| 애완슬라임 | Lv.3 | 던전 귀환 시 자원 자동 수집. 레벨 상승 시 수집량 증가 |
| 무기 제작소 | Lv.5 | 무기 아이템 제작 |
| 방어구 제작소 | Lv.5 | 방어구 아이템 제작 |
| 연금술 실험실 | Lv.5 | 소모품 및 연금술 아이템 제작 |
| 주문서 제작소 | Lv.5 | 강화·수리·속성 부여 주문서 제작 |
| 땜장이 작업소 | Lv.5 | 장비 수리 및 개조 |
| 정화의 제단 | Lv.3 | 오염 아이템 정화 · 저주 해제 |
| 아이템 합성소 | Lv.5 | 아이템 합성으로 더 강력한 아이템 생성 |
| 레시피 보관함 | 없음 | 인게임 획득 레시피 자동 보관 · 레벨업 없음 |

### 시설 상태

- **미개방**: 개방 비용(재료) 표시 + [개방하기] 버튼
- **Lv.N**: 기능 이용 가능 + 업그레이드 영역 표시
- **최대 레벨**: 업그레이드 버튼 비활성, "최대 레벨 달성" 표시

### 업그레이드 흐름

1. 상세 패널 하단 [업그레이드] 버튼 클릭
2. 팝업 오픈 — 필요 재료 / 보유량 비교 표시
3. 재료 충분 시 업그레이드 버튼 활성 → 클릭 시 레벨 상승
4. 재료 부족 시 업그레이드 버튼 비활성

### 애완슬라임 수집 자원 풀

| 레벨 | 수집 종류 수 | 수집 가능 자원 |
|------|------------|--------------|
| Lv.1 | 2종 | 고철 조각 · 유기물 · 목재 |
| Lv.2 | 3종 | 고철 조각 · 철 스크랩 · 유기물 · 화학 약품 |
| Lv.3 | 4종 | 철 스크랩 · 강화 부품 · 화학 약품 · 강화 소재 |

## 4. 보관소 (HUD 오버레이)

진입: ZONE-E 보관소 버튼 → HUD 오버레이

### 패널 구성 순서 (좌 → 우)

| 순서 | 패널 | 설명 |
|------|------|------|
| 1 | 로드아웃 프리셋 | 프리셋 목록 (활성 프리셋 강조 표시) |
| 2 | 필터링 아이콘 | 보관함 필터 세로 아이콘 바 |
| 3 | 보관함 | 기본 60칸, 확장 가능 |
| 4 | 장비 슬롯 + 배낭 패널 | 상단: 장비 슬롯 / 하단: 배낭 패널 |

### 4-1. 로드아웃 프리셋

프리셋 목록을 표시하며, 활성 프리셋은 강조 색상으로 구분한다.
- 프리셋 #1 (활성) / 프리셋 #2 / + 프리셋 추가

### 4-2. 보관함 필터링

보관함 왼쪽에 **세로 아이콘 바**로 배치. 마우스 오버 시 라벨이 아이콘 오른쪽에 표시.

| 구분 |
|------|
| 전체 |
| 무기 |
| 방어구 |
| 소모성 |
| 레시피 |
| 기타 |

### 4-3. 보관함

| 항목 | 내용 |
|------|------|
| 기본 슬롯 수 | 60칸 (4열 그리드, 세로 스크롤) |
| 확장 단위 | +24칸 |
| 확장 비용 | 골드 (1,000 + 확장 횟수 x 500 G) — 1회: 1,000 / 2회: 1,500 / 3회: 2,000 ... |
| 확장 방법 | 헤더 [+ 확장] 버튼 → 확장 팝업에서 비용 확인 후 확정 |
| 골드 부족 시 | 확장하기 버튼 비활성 |

### 4-4. 장비 슬롯

4열 그리드 배치. 배낭 슬롯은 하단에 단독 전 너비 행으로 표시.

| 슬롯 | 비고 |
|------|------|
| 주무기 | — |
| 보조무기 | — |
| 헬멧 | — |
| 상의 | — |
| 하의 | — |
| 장갑 | — |
| 신발 | — |
| 액세서리 | — |
| 배낭 | 장착 배낭명 표시. 배낭에 따라 배낭 패널 슬롯 수 변동 |

### 4-5. 배낭 패널

장비 슬롯 하단에 배치. 장착된 배낭 종류에 따라 그리드 크기 변동.

| 배낭 종류 | 그리드 | 슬롯 수 |
|----------|--------|--------|

| 중형 배낭 | 4x3 | 12칸 |

## 8. 매치메이킹 예외 처리

| 상황 | 처리 방식 |
|---|---|
| 파티원 중 IN_GAME 멤버 존재 | 플레이 버튼 비활성 + "진행 중인 레이드가 있는 파티원이 있습니다" 툴팁 |
| 선택 던전 서버 점검 중 | 핀 비활성 + "점검 중" 레이블 |
| 매치메이킹 타임아웃 (120초) | 자동 취소 + "매치를 찾지 못했습니다" 토스트 |
| 네트워크 연결 끊김 | 매치메이킹 즉시 중단 + 재연결 시도 모달 |
| 로드아웃 무기 미장착 | 경고 팝업 "장비 없이 출격하시겠습니까?" |
| 보관함 공간 0% | 경고 팝업 "보관함 공간이 가득 찼습니다." |
| 보관소 공간 부족 (결과창) | 이동 실패 · 에러 토스트 |
| 작업장 업그레이드 재료 부족 | 업그레이드 버튼 비활성 |
