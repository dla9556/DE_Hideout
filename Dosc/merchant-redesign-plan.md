# 교환소/상인 UI 재설계 계획서

> 작성일: 2026-03-12 | 기반: 인터뷰 확정 답변

---

## 목표

기존 3단 패널(카테고리 사이드바 + 아이템 그리드 + 상세) 구조를 →
**SVG 암거래 골목 씬 + 상인 노점 자유 배치** 방식으로 전면 재설계한다.

---

## 인터뷰 확정 답변

| 항목 | 결정 |
|------|------|
| 상인 수 | 7개 전부 배치 |
| 분위기 | 중세 판타지 암거래 골목 — 보라/초록 가스등, 석판 골목, 의심스러운 인물 |
| 스탠드 스타일 | 천으로 가려진 노점 부스 (체크 패턴 어닝 + 간판) |
| 아이템 표시 | 3열 그리드 (아이콘 대형 + 이름 + 가격), 상세 뷰 우측에 대형 NPC 컬럼 추가 |
| 상인별 배경 | 각 상인마다 다른 배경 CSS 아트 |
| 구매 방식 | 사전 확인 팝업 → hold 1초 → 구매 완료 확인창 |

## 컴포넌트 구조

```
#hp-merchant
├── #mc-alley              (골목 뷰 — 기본 표시)
│   ├── #mc-alley-svg      (SVG 암거래 씬 배경, z-index:0)
│   ├── .mc-alley-name     (골목 이름 오버레이, z-index:5)
│   └── #mc-alley-stalls   (상인 스탠드 컨테이너, z-index:2)
│       └── .mc-stall      (절대 위치, MC_LAYOUT 기반)
│           ├── .mc-stall-awning  (체크 패턴 어닝 + 간판)
│           └── .mc-stall-body   (아이콘 + 상인명)
├── #mc-merchant-view      (상인 상세 뷰 — 진입 시 표시, display:flex)
│   ├── #mc-mer-nav        (좌측 빠른 이동 패널, 185px, border-right)
│   ├── #mc-mer-left       (중앙 상인 콘텐츠, flex:1)
│   │   ├── #mc-mer-bg       (배경 CSS 아트)
│   │   ├── .mc-mer-back     (← 교환소 골목으로 버튼)
│   │   └── #mc-mer-body     (가로 2-column 레이아웃)
│   │       ├── #mc-mer-items-col  (3열 아이템 그리드, flex:1)
│   │       └── #mc-mer-npc-col   (대형 NPC 컬럼, 130px, align:flex-end)
│   └── #mc-buy-panel      (우측 구매 패널, 210px, 아이템 클릭 시 표시)
├── #mc-buy-confirm        (사전 확인 모달 — "정말 구매하시겠습니까?")
├── #mc-hold-modal         (hold-to-buy 모달 — 1초 누르기)
└── #mc-buy-result         (구매 완료 확인창)
```

|----|--------|--------|-------------|----------|
| general | 잡화 상인 | 🛍 | 기본 (황갈색) | 왼쪽 top 4% |

## SVG 씬 구성 요소 (암거래 골목)

| 요소 | 설명 |
|------|------|
| 배경 rect | fill #010103 (어두운 보라흑) |
| 왼/오른쪽 벽면 polygon | 석판 질감 (보라회색 선) |
| 바닥 polygon | 습한 조약돌 (원근 격자) |
| 소실점 글로우 | radialGradient 보라색, cx 47% cy 20% |
| 가스등 4개 | 초록-노랑 glow, CSS .mc-gl1~4 깜박임 |
| 중앙 랜턴 | 보라 빛 천장 매달림 |
| 수배 포스터 2개 | 왼쪽 벽 사각형 |
| 의심스러운 인물 2개 | 소실점 근처 후드 실루엣 |
| 밀수품 상자/통 | 좌/우 바닥 |
| 천장 체인 2개 | 느슨하게 매달림 |
| 하단 보라 안개 | linearGradient 덮개 |
| 비네트 | 상하단 어두움 처리 |

## 구매 UX — 사전 확인 + Hold-to-Buy

| 단계 | 동작 |
|------|------|
| 아이템 클릭 | 우측 구매 패널 오픈 (아이템 정보 + 수량 + 구매 버튼) |
| "구매하기" 클릭 | 사전 확인 팝업 ("정말 구매하시겠습니까?" + 가격 표시) |
| 팝업 "확인" | hold-to-buy 모달 표시 |
| 버튼 1초 누르기 | 진행 바 채워짐 → 구매 완료 |
| 손 뗌 | 취소 — 진행 바 리셋 |
| 구매 완료 | 아이콘 + 아이템명 + 수량 확인창 |
| 확인창 닫기 | Enter / ESC / 확인 버튼 |
