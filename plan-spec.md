# 구조도 작성 계획서 (plan-spec)

> 작성일: 2026-03-10 | 최종 수정: 2026-03-12

## 목표

D&E 아웃게임의 **메인 순환 루프**와 **메뉴별 세부 항목**을 시각적 구조도로 정리한다.
개발팀 공유 및 설계 기준 문서로 활용한다.

---

|------|------|
| 표현 방식 | Mermaid 문법 (flowchart/graph) |

## 구조도 구성 항목

### 1. 메인 순환 루프

유저가 반복하는 핵심 행동 흐름:

```
로그인 → 캐릭터 선택 → 메인 허브(은거지) → [씬 이동] → 메인 허브 복귀
                                                ↓
                                         플레이 진입 루프
                         지역 맵 던전 선택 → 로드아웃 → 매치메이킹 → 인게임 → 결과창 → 복귀
```

### 2. 메뉴별 세부 항목 (Zone A~E)

| Zone | 구성 요소 | 이동 대상 |
|------|----------|----------|
| ZONE-A (상단) | 은거지, 닉네임+Lv, 공방, 성장, 교환소/상인, 길드 사무소`[미구현]`, 용병`[미구현]`, 우편함`[미구현]`, 소식`[미구현]`, 시스템, 재화(실버·D토큰·계곡의 인장), 서버상태 | 각 HUD 오버레이 |
| ZONE-B (좌측) | 길드 사무소 위젯`[미구현]`, 채팅`[미구현]` | — |
| ZONE-C (중앙) | 캐릭터 렌더(화면 가득), 닉네임·레벨(렌더 상단), 스쿼드 표시 | — |
| ZONE-D (우측) | 스쿼드 설정(랜덤 스쿼드 포함), 플레이 버튼 | 지역 맵 던전 선택 오버레이 |
| ZONE-E (하단·항상표시) | 보관소, 캐릭터 변경, 친구`[미구현]` | 보관소 HUD, 캐릭터 선택 |

### 3. 플레이 진입 서브플로우

```
플레이 버튼
  → 지역 맵 던전 선택 오버레이 (핀 방식)
    → 던전 상세 패널
      → 로드아웃 선택 (무료/커스텀)
        → 매치메이킹 모달 (PvEvP / PvE전용)
          → 매치메이킹 진행 중
            ├─ 성공 → 로딩 화면 → 인게임 → 결과창 → 복귀
            ├─ 취소 → 지역 맵 선택
            └─ 타임아웃(120s) → 허브 복귀
```

### 4. 공방 시설

> 최종 수정: 2026-03-12 — 골목 뷰 + 시설 상세 뷰 이중 구조로 전면 재설계

```
공방 HUD
  → 골목 뷰: SVG 원근 씬 + 10개 시설 문 자유 배치 (좌/우 벽면)
       기본 작업대 · 방어구 제작소 · 땜장이 작업소 · 레시피 보관함 · 무기 제작소 (왼쪽)
       애완슬라임 · 정화의 제단 · 연금술 실험실 · 아이템 합성소 · 주문서 제작소 (오른쪽)
    → 시설 문 클릭 → 시설 상세 뷰:
        좌측 빠른 이동 패널 | 중앙 시설 내용 | 우측 제작 패널
        미개방: 🔒 표시, 클릭 시 상세에서 개방 가능
        개방됨: 시설별 기능 + 업그레이드 영역
          → 업그레이드 팝업: 재료 보유량 확인
              재료 충분 → [업그레이드] 버튼 활성
              재료 부족 → [업그레이드] 버튼 비활성
        레시피 클릭 → 제작 패널:
          수량 설정 → 꾸욱 눌러 제작 (2초 hold)
          → 제작 완료 확인창 (Enter/ESC/버튼 닫기)
```

### 5. 교환소/상인 (Merchant) HUD

> 최종 수정: 2026-03-12 — 암거래 골목 SVG 씬 + 자유 배치 노점 구조로 전면 재설계

```
교환소/상인 HUD
  → 골목 뷰: SVG 암거래 골목 씬 (보라-흑 분위기, 가스등 깜박임)
       7개 노점 자유 배치: 무기 · 방어구 · 소모품 · 마법용품 · 암거래 · 특수거래 · 신비로운 노점
       각 노점에 움직이는 NPC (좌우 왕복 애니메이션)
    → 노점 클릭 → 상인 상세 뷰:
        좌측 빠른 이동 패널 | 중앙 아이템 그리드(3열, 대형) + 우측 NPC(대형) | 우측 구매 패널
        아이템 카드 클릭 → 구매 패널 활성:
          수량 설정 (−/+)
          잔액 충분 → [구매하기] 버튼 활성
          잔액 부족 → [구매하기] 버튼 비활성
          → 사전 확인 팝업 → 꾸욱 눌러 구매 (1초 hold)
          → 구매 완료 확인창 (Enter/ESC/버튼 닫기)
```

### 6. 결과창

```
인게임 종료
  → 결과창 (획득 정보 표시)
    → [획득한 아이템 보관소로 이동]
        보관소 공간 있음 → 이동 완료 토스트
        보관소 공간 부족 → 에러 토스트
    → [복귀] → 허브
    → [탈출 성공 · 소지품 유지 → 허브]
    → [탈출 실패 · 소지품 소실 → 허브]
```

### 7. 보관소 HUD

```
보관소 HUD (패널 순서: 로드아웃 프리셋 | 필터 아이콘 | 보관함 | 장비 슬롯+배낭)

보관함:
  기본 60칸 (4열 그리드, 세로 스크롤)
  → [+ 확장] 버튼 → 확장 팝업
      보유 실버 ≥ 비용 → 확장하기 버튼 활성 → 클릭 시 +24칸
      보유 실버 < 비용 → 확장하기 버튼 비활성
  확장 비용: 1,000 + (확장 횟수 × 500) 실버

필터링 아이콘 바 (세로):
  전체 / 무기 / 방어구 / 소모성 / 레시피 / 기타
  마우스 오버 시 라벨이 아이콘 오른쪽에 표시

장비 슬롯:
  주무기 · 보조무기 · 헬멧 · 상의 · 하의 · 장갑 · 신발 · 액세서리
  배낭 (하단 단독 행)

배낭 패널 (장비 슬롯 하단):
  기본 배낭 4×2 (8칸)
  중형 배낭 4×3 (12칸)
  대형 배낭 4×4 (16칸)
  특대형 배낭 4×5 (20칸)
  장착 없음 → 빈 상태 메시지 표시

  [모두 보관함으로 이동] 버튼 (패널 헤더 우측):
    배낭 아이템 없음 → 안내 토스트
    보관함 여유 공간 부족 → 차단 + 에러 토스트 (필요/여유 칸 표시)
    이동 성공 → 배낭 비움 + 성공 토스트

안전 가방 슬롯 (배낭 패널 하단 · 4열 그리드 셀 크기):
  기본·중형 배낭 → 1칸
  대형·특대형 배낭 → 2칸
  장착 없음 → 0칸 (섹션 표시 유지)
```
