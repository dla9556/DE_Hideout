## 목표

D&E 아웃게임의 **메인 순환 루프**와 **메뉴별 세부 항목**을 시각적 구조도로 정리한다.
개발팀 공유 및 설계 기준 문서로 활용한다.

---

## 인터뷰 확정 답변

| 항목 | 결정 |
|------|------|
| 표현 방식 | Mermaid 문법 (flowchart/graph) |
| 표현 깊이 | 씬(Scene) + 주요 UI 컴포넌트 (Zone 수준) |
| 루프 기준 | 유저 행동 기준 (로그인 → 허브 → 씬 이동 → 허브 복귀) |
| 미구현 항목 | `[미구현]` 태그 포함, 본문에 함께 표기 |
| 저장 위치 | 프로젝트 루트 (`structure.md`) |

## 구조도 구성 항목

### 1. 메인 순환 루프

유저가 반복하는 핵심 행동 흐름:

```
로그인 → 캐릭터 선택 → 메인 허브(은거지) → [씬 이동] → 메인 허브 복귀
                                                ↓
                                         플레이 진입 루프
                         지역 맵 던전 선택 → 로드아웃 → 매치메이킹 → 인게임 → 결과창 → 복귀
```

### 2. 메뉴별 세부 항목 (Zone A~E)

| Zone | 구성 요소 | 이동 대상 |
|------|----------|----------|
| ZONE-A (상단) | 은거지, 닉네임+Lv, 퀘스트`[미구현]`, 작업장, 상인, 성장, 성장 트리, 우편함`[미구현]`, 설정, 재화, 서버상태 | 각 HUD 오버레이 |
| ZONE-B (좌측) | 퀘스트 추적기`[미구현]`, 채팅`[미구현]` | — |
| ZONE-C (중앙) | 캐릭터 렌더(화면 가득), 닉네임·레벨(렌더 상단), 스쿼드 표시 | — |
| ZONE-D (우측) | 스쿼드 설정(랜덤 스쿼드 포함), 플레이 버튼 | 지역 맵 던전 선택 오버레이 |
| ZONE-E (하단·항상표시) | 인벤토리, 캐릭터 변경, 친구`[미구현]` | 인벤토리 HUD, 캐릭터 선택 |

### 3. 플레이 진입 서브플로우

```
플레이 버튼
  → 지역 맵 던전 선택 오버레이 (핀 방식)
    → 던전 상세 패널
      → 로드아웃 선택 (무료/커스텀)
        → 매치메이킹 모달 (PvEvP / PvE전용)
          → 매치메이킹 진행 중
            ├─ 성공 → 로딩 화면 → 인게임 → 결과창 → 복귀
            ├─ 취소 → 지역 맵 선택
            └─ 타임아웃(120s) → 허브 복귀
```

### 4. 작업장 시설

```
작업장 HUD
  → 사이드바: 5개 시설 목록 (상태 뱃지 표시)
    → 상세 패널:
        미개방: 개방 비용 + [개방하기]
        개방됨: 시설별 기능 + 업그레이드 영역
          → 업그레이드 팝업: 재료 보유량 확인
              재료 충분 → [업그레이드] 버튼 활성
              재료 부족 → [업그레이드] 버튼 비활성
```

### 5. 결과창

```
인게임 종료
  → 결과창 (획득 정보 표시)
    → [획득한 아이템 인벤토리로 이동]
        인벤토리 공간 있음 → 이동 완료 토스트
        인벤토리 공간 부족 → 에러 토스트
    → [복귀] → 허브
    → [탈출 성공 · 소지품 유지 → 허브]
    → [탈출 실패 · 소지품 소실 → 허브]
```

### 6. 인벤토리 HUD

```
인벤토리 HUD (패널 순서: 로드아웃 프리셋 | 필터 아이콘 | 보관함 | 장비 슬롯+배낭)

보관함:
  기본 60칸 (4열 그리드, 세로 스크롤)
  → [+ 확장] 버튼 → 확장 팝업
      보유 골드 ≥ 비용 → 확장하기 버튼 활성 → 클릭 시 +24칸
      보유 골드 < 비용 → 확장하기 버튼 비활성
  확장 비용: 1,000 + (확장 횟수 × 500) 골드

필터링 아이콘 바 (세로):
  전체 / 무기 / 방어구 / 소모성 / 레시피 / 기타
  마우스 오버 시 라벨이 아이콘 오른쪽에 표시

장비 슬롯:
  주무기 · 보조무기 · 헬멧 · 상의 · 하의 · 장갑 · 신발 · 액세서리
  배낭 (하단 단독 행)

배낭 패널 (장비 슬롯 하단):
  기본 배낭 4×2 (8칸)
  중형 배낭 4×3 (12칸)
  대형 배낭 4×4 (16칸)
  특대형 배낭 4×5 (20칸)
  장착 없음 → 빈 상태 메시지 표시
```
